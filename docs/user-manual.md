# 사용자 매뉴얼

## 철스크랩 수급 계획 및 구매 관리 시스템

**동국제강 주식회사 원료기획팀**
**시스템 버전**: 1.0
**최종 수정일**: 2026년 3월

---

## 1. 시스템 개요

본 시스템은 철스크랩 원료의 수급 계획 수립부터 구매 실적 관리까지 전 과정을 하나의 화면에서 처리할 수 있는 웹 기반 관리 도구입니다.

### 1.1 주요 기능

| 기능 | 설명 |
|------|------|
| 수급계획 | 연간/월간 수급 계획 수립 및 실적 대비 관리 |
| 공장배분 | 인천/포항 공장별 물량 배분 및 달성률 추적 |
| 발주관리 | 거래처별 발주 등록, 조회, 발주서 출력 |
| 거래처관리 | 거래처 정보 등록 및 납품 실적 현황 조회 |
| 수입계약 | 수입 스크랩 계약 등록 및 진행 상태 추적 |
| 재고현황 | 공장별/등급별 재고 수준 모니터링 |
| 구매실적 | 기간별 구매량/금액 통계 및 CSV 내보내기 |

### 1.2 접속 환경

- **권장 브라우저**: Chrome, Edge (최신 버전)
- **화면 해상도**: 1280×720 이상 권장
- **인터넷 연결**: 차트 표시를 위해 필요

---

## 2. 로그인

### 2.1 로그인 방법

1. 시스템 URL에 접속하면 로그인 화면이 나타납니다.
2. 아이디와 비밀번호를 입력합니다.
3. **로그인** 버튼을 클릭합니다.
4. 인증 성공 시 대시보드 메인 화면으로 이동합니다.

### 2.2 로그인 오류 시

- 아이디 또는 비밀번호가 틀리면 빨간색 오류 메시지가 표시됩니다.
- 비밀번호 입력란이 초기화되며, 다시 입력할 수 있습니다.

### 2.3 로그아웃

- 화면 우측 상단의 **로그아웃** 버튼을 클릭합니다.
- 로그아웃 후 로그인 화면으로 돌아갑니다.
- 브라우저 탭을 모두 닫아도 세션이 종료됩니다.

---

## 3. 화면 구성

### 3.1 공통 레이아웃

- **상단 헤더**: 시스템명, 현재 날짜, 로그인 사용자명, 로그아웃 버튼
- **탭 네비게이션**: 7개 업무 영역을 탭으로 전환
- **본문 영역**: 선택된 탭의 KPI 카드, 차트, 테이블, 입력 폼

### 3.2 탭 전환

- 상단 탭 버튼을 클릭하면 해당 화면으로 즉시 전환됩니다.
- 활성 탭은 하단에 주황색 선으로 표시됩니다.

---

## 4. 수급계획

### 4.1 화면 구성

- **KPI 카드**: 총 계획량, 국산/수입 비율, 평균 달성률
- **비율 설정 슬라이더**: 국산/수입 비율 및 월간 목표 물량 조정
- **월별 계획 차트**: 계획 대비 실적 막대 차트
- **월간 수급 계획표**: 12개월 계획/국산/수입/실적 데이터

### 4.2 데이터 편집

#### 셀 직접 편집
1. 테이블의 편집 가능한 셀을 **더블클릭**합니다.
2. 셀이 노란색 입력 모드로 변경됩니다.
3. 숫자를 입력한 후 **Enter** 키를 누르거나 셀 바깥을 클릭합니다.
4. **Esc** 키를 누르면 편집을 취소합니다.
5. 저장 시 KPI와 차트가 자동으로 갱신됩니다.

#### 엑셀 붙여넣기
1. 테이블 영역을 클릭하여 포커스를 줍니다 (파란색 테두리 표시).
2. 엑셀에서 데이터를 복사합니다 (월, 계획, 국산, 수입, 실적 순).
3. **Ctrl+V**로 붙여넣습니다.
4. 최대 12개월 데이터를 한 번에 입력할 수 있습니다.
5. 헤더 행이 포함되어 있어도 자동으로 건너뜁니다.

#### 초기화
- **초기화** 버튼을 클릭하면 기본 데이터로 되돌립니다.
- 확인 팝업에서 승인해야 초기화가 실행됩니다.

### 4.3 주의사항

- 계획, 국산, 수입 물량은 0 이상의 숫자만 입력 가능합니다.
- 실적은 빈 값(미입력)이 허용됩니다.
- 편집된 데이터는 브라우저에 자동 저장되며, 같은 브라우저로 재접속 시 유지됩니다.

---

## 5. 공장배분

### 5.1 화면 구성

- **KPI 카드**: 인천/포항 배분 비율, 최근월 입고량 및 달성률
- **공장별 현황 카드**: 달성률 진행 바 + 도넛 차트
- **월별 배분 실적표**: 인천/포항 계획·실적·달성률

### 5.2 데이터 편집

- 수급계획과 동일한 방식으로 셀 편집 및 엑셀 붙여넣기가 가능합니다.
- 붙여넣기 시 컬럼 순서: 월, 인천 계획, 인천 실적, 포항 계획, 포항 실적
- 달성률은 자동 계산됩니다.

---

## 6. 발주관리

### 6.1 화면 구성

- **KPI 카드**: 발주 건수, 발주 물량, 완료/진행 건수 (실제 데이터 기반 자동 계산)
- **발주 목록 테이블**: 발주번호, 거래처, 등급, 수량, 단가, 금액, 공장, 날짜, 상태

### 6.2 신규 발주 등록

1. **+ 신규 발주** 버튼을 클릭합니다.
2. 모달 창에서 아래 항목을 입력합니다.
   - 거래처 (선택)
   - 등급 (선택)
   - 수량 (톤)
   - 매입단가 (원/톤)
   - 납품공장 (인천/포항)
   - 납품일
3. **발주 등록** 버튼을 클릭합니다.
4. 등록된 발주는 목록 최상단에 추가되며, KPI가 자동 갱신됩니다.

### 6.3 발주서 출력

1. **PDF 발주서 출력** 버튼을 클릭합니다.
2. 가장 최근 발주 내용이 발주서 양식으로 미리보기 됩니다.
3. **인쇄 / PDF 저장** 버튼으로 인쇄하거나 PDF로 저장할 수 있습니다.

---

## 7. 거래처관리

### 7.1 화면 구성

- **거래처 목록 테이블**: 코드, 거래처명, 지역, 대표자, 연락처, 공급능력, 납품량, 신뢰등급
- **납품 비중 차트**: 거래처별 납품 비중 파이 차트
- **납품 추이 차트**: 주요 거래처 월별 납품 추이 라인 차트

### 7.2 거래처 등록

1. **+ 거래처 등록** 버튼을 클릭합니다.
2. 거래처명, 대표자, 지역, 연락처, 월 공급능력을 입력합니다.
3. **등록** 버튼을 클릭하면 목록과 차트에 즉시 반영됩니다.
4. 거래처 코드(S001, S002...)는 자동 부여됩니다.

---

## 8. 수입계약

### 8.1 화면 구성

- **KPI 카드**: 계약 건수, 수입 물량, 평균 CFR 가격, 적용 환율 (실제 데이터 기반)
- **계약 목록 테이블**: 계약번호, 수출국, 공급사, 등급, 수량, CFR, 환율, 선적일, 도착예정일, 상태
- **진행 현황 차트**: 상태별(계약/선적/운송중/도착) 건수 막대 차트

### 8.2 신규 계약 등록

1. **+ 신규 계약** 버튼을 클릭합니다.
2. 수출국, 공급사, 등급, 수량, CFR 가격, 적용 환율, 선적일, 도착 예정일을 입력합니다.
3. **계약 등록** 버튼을 클릭하면 목록, KPI, 차트가 모두 갱신됩니다.

---

## 9. 재고현황

### 9.1 화면 구성

- **KPI 카드**: 전체 재고량, 인천/포항 야적장 재고, 평균 회전일수
- **공장별 등급 차트**: 인천/포항 공장의 현재 재고 vs 적정 재고 막대 차트
- **재고 상세 테이블**: 공장, 등급, 현재 재고, 적정 재고, 재고율, 상태, 수준 바

### 9.2 상태 판단 기준

| 재고율 | 상태 | 색상 |
|--------|------|------|
| 110% 이상 | 과잉 | 주황 |
| 85% ~ 110% | 적정 | 초록 |
| 85% 미만 | 부족 | 빨강 |

## 10. 구매실적

### 10.1 화면 구성

- **KPI 카드**: 총 구매량, 총 구매금액, 평균 매입단가, 거래처 수
- **월별 구매량 차트**: 구매량(막대) + 금액(라인) 복합 차트
- **등급별 비중 차트**: 등급별 구매 비중 도넛 차트
- **월별 실적 테이블**: 등급별 구매량 및 합계

### 10.2 데이터 내보내기

- **엑셀 내보내기** 버튼: CSV 파일로 다운로드 (파일명: `구매실적_2026.csv`)
- **인쇄** 버튼: 브라우저 인쇄 화면 실행

## 11. 데이터 저장 안내

### 11.1 자동 저장 항목

아래 데이터는 편집 즉시 브라우저에 자동 저장되며, 같은 브라우저로 재접속 시 유지됩니다.

- 수급계획 (월간 계획/실적 데이터)
- 공장배분 (인천/포항 계획/실적 데이터)
- 발주 목록
- 거래처 목록
- 수입계약 목록

### 11.2 저장되지 않는 항목

- 재고현황 (고정 데이터)
- 구매실적 (고정 데이터)
- 슬라이더 설정값

### 11.3 주의사항

- 데이터는 현재 사용 중인 **브라우저에만** 저장됩니다.
- 다른 PC, 다른 브라우저에서는 데이터가 공유되지 않습니다.
- 브라우저의 캐시/데이터를 삭제하면 저장된 데이터가 초기화됩니다.
- 로그인 세션은 브라우저 탭을 모두 닫으면 종료됩니다.

## 12. 자주 묻는 질문 (FAQ)

**Q. 데이터를 다른 PC에서도 보고 싶습니다.**
A. 현재는 브라우저 저장 방식이라 PC 간 공유가 불가합니다. 구매실적의 CSV 내보내기를 활용하시거나, 향후 서버 연동 시 가능해집니다.

**Q. 엑셀 붙여넣기가 안 됩니다.**
A. 테이블 영역을 먼저 클릭하여 파란색 테두리가 나타나는지 확인하세요. 이후 Ctrl+V를 사용합니다.

**Q. 차트가 표시되지 않습니다.**
A. 인터넷 연결을 확인하세요. 차트 라이브러리(Chart.js)는 CDN에서 불러옵니다.

**Q. 초기화 후 데이터를 복구할 수 있나요?**
A. 초기화 후에는 복구할 수 없습니다. 초기화 전에 반드시 확인 팝업을 확인하세요.

**Q. 발주서를 실제 PDF 파일로 저장하려면?**
A. PDF 발주서 출력 화면에서 인쇄 / PDF 저장 버튼을 누른 후, 프린터를 "PDF로 저장"으로 선택하세요.
