## 탭 구성 한눈에 보기

| 탭 | 핵심 기능 | 데이터 저장 |
|----|----------|------------|
| 수급계획 | 월별 계획/실적 관리, 국산·수입 비율 조정 | 자동 저장 |

| 거래처관리 | 거래처 등록, 납품 실적 조회 | 자동 저장 |

| 구매실적 | 구매 통계, CSV 내보내기 | 고정 데이터 |

## 데이터 편집 (수급계획 / 공장배분)

### 셀 편집
1. 셀 **더블클릭** → 노란색 입력 모드
2. 숫자 입력 → **Enter** 저장 / **Esc** 취소

### 엑셀 붙여넣기
1. 테이블 클릭 (파란 테두리 확인)
2. 엑셀에서 복사 → **Ctrl+V**
3. 최대 12개월, 헤더 자동 건너뜀

### 초기화
- **초기화** 버튼 → 확인 팝업 승인 → 기본값 복원

---

## 신규 등록

### 발주 등록
**+ 신규 발주** → 거래처·등급·수량·단가·공장·납품일 입력 → **발주 등록**

### 거래처 등록
**+ 거래처 등록** → 거래처명·대표자·지역·연락처·공급능력 입력 → **등록**

### 수입계약 등록
**+ 신규 계약** → 수출국·공급사·등급·수량·CFR·환율·선적일·도착일 입력 → **계약 등록**

---

## 발주서 출력

1. 발주관리 탭 → **PDF 발주서 출력**
2. 최근 발주 내용 미리보기
3. **인쇄 / PDF 저장** → 프린터를 "PDF로 저장" 선택

---

## 데이터 내보내기

- 구매실적 탭 → **엑셀 내보내기** → CSV 파일 다운로드
- 구매실적 탭 → **인쇄** → 브라우저 인쇄

## 재고 상태 기준

| 재고율 | 상태 | 색상 |
|--------|------|------|
| 110% 이상 | 과잉 | 🟠 주황 |
| 85~110% | 적정 | 🟢 초록 |
| 85% 미만 | 부족 | 🔴 빨강 |

---

## 주의사항

- 데이터는 **현재 브라우저에만** 저장됩니다 (다른 PC/브라우저 공유 불가).
- 브라우저 캐시 삭제 시 데이터가 초기화됩니다.
- 차트 표시를 위해 **인터넷 연결**이 필요합니다.
- 모든 탭을 닫으면 로그인 세션이 종료됩니다.

## 문제 해결

| 증상 | 조치 |
|------|------|
| 로그인 반복 | 재로그인 |
| 빈 화면 | F12 → Console 오류 확인 |
| 차트 안 보임 | 인터넷 연결 확인 |
| 데이터 안 보임 | 초기화 버튼 또는 F12 → Application → Local Storage 삭제 후 새로고침 |
| 변경 미반영 | Ctrl+Shift+R (캐시 무시 새로고침) |
